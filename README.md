
# СКАМ ТАРО ✨ Professional Build

Инструкция по созданию полноценного EXE файла.

## 1. Подготовка
1. Установите [Node.js](https://nodejs.org/) (версия LTS).
2. Выполните в терминале (CMD) папке проекта:
```cmd
npm install
```

## 2. Настройка API ключа
Создайте файл `.env` в корневой папке (если его нет) и добавьте туда ваш ключ:
```env
API_KEY=ваш_ключ_здесь
```

## 3. Сборка EXE
Теперь сборка выполняется одной официальной командой:
```cmd
npm run build:exe
```

### Что произойдет:
1. Vite скомпилирует React-приложение в папку `dist`.
2. `electron-builder` упакует содержимое `dist` и движок в один файл.
3. Готовый файл появится в папке **`release`** под названием `ScamTarot.exe`.

## 4. Почему этот метод лучше?
- **Портативность:** Получается один файл, который можно перекинуть на флешку.
- **Оффлайн:** Работает без локального сервера Vite.
- **Чистота:** Никаких лишних консольных окон при запуске программы.
